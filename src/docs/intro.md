# NutUI-Biz

NutUI-Biz 组件库，适用于 React 技术栈的 H5 电商应用，开箱即用，帮助研发快速开发用户界面，提升开发效率，改善开发体验。

![](https://img12.360buyimg.com/imagetools/jfs/t1/93525/10/35630/3509/63e5e07fFee1229de/364b5180962e277c.png)

##  特性

* 🚀 20+ 高质量组件，覆盖移动端主流场景
* 📖 基于京东 APP 10.0 视觉规范
* 🍭 支持按需引用
* 📖 详尽的文档和示例
* 💪 支持 TypeScript
* 🌍 支持国际化
* 🍭 单元测试覆盖率超过 80%，保障稳定性

## 版本

* **版本：** @nutui/nutui-biz
* **支持的框架：** React 17\18
* **视觉规范：** 京东 APP 10.0 规范
* **兼容：** 现代浏览器以 Chrome >= 51、iOS >= 10.0
